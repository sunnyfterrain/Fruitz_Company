# 프로젝트 소개
- 국내 커피 판매 사이트 [프릳츠커피](https://fritz.co.kr/) 클론 코딩
### 프로젝트 선정 이유
- 깔끔하고 심플한 UI
- 시간이 많이 들어가는 디자인이 많지 않아 기능 구현에 집중 가능
- 프론트엔드 개발을 위한 필수적인 기능이지만 구현하기에 크게 어렵지 않음
